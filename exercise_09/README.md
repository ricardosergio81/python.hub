#What will be the Output?

```
a = True
b = False
print(a or a and b and a)
```

Options:
- [X] True

- [ ] False

- [ ] Error

- [ ] None of above


##Alternative
```
a = True
b = False
print(a or (a and b and a))
```
```
True
```

```
a = True
b = False
print((a or a) and b and a)
```
```
False
```
